# 05 — Prompt Engineering for ADA

## Principles

### Goal-Based Prompting
Avoid telling ADA how to code.

### Use Examples
Show format expectations.

### Iterate
ADA self-corrects well.

### Constrain
- "Keep it simple."
- "Minimize code."

### Combine reasoning + execution
"Explain briefly, then run Python."

### Decompose tasks
Exploration → Cleaning → Transformation → Output

### Provide Context
Describe columns and domain assumptions.

### Debugging
- Inspect tracebacks
- Verify assumptions
