# 07 — Problem Selection and Tactics

## Good Fits
- small data
- iterative tasks
- analysis + visualization
- data wrangling
- rapid prototypes

## Poor Fits
- large data (>100 MB)
- GPU tasks
- heavy APIs
- long jobs

## Choosing Good Problems
Check:
- data size
- iteration need
- clarity
- Python suitability

## Strategy
Inspect → Plan → Execute → Fix → Iterate

## Red Flags
- unclear instructions
- missing examples
- ambiguous data
