## What ADA Is
Advanced Data Analysis (formerly Code Interpreter) provides:
- a sandboxed Python environment
- temporary filesystem
- ability to inspect, transform, and visualize data

## Why It Exists
ChatGPT alone cannot:
- run code
- inspect files
- validate outputs

ADA fills this gap.

## Capabilities
- Data analysis
- File transformations
- Visualization
- Rapid prototyping
- Micro-utilities

## Limitations
- No persistence
- No heavy compute
- No GPU
- No long-running jobs

## Workflow
1. Upload files
2. ADA inspects them
3. Writes Python
4. Executes
5. Self-corrects
6. Iterates

## Best Uses
- EDA
- Reporting
- Cleaning
- Conversion
- Small tools
